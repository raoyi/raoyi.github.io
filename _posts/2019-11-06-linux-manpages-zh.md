**Debian / Ubuntu:**

```
sudo apt update
sudo apt install manpages-zh
```

**Arch Linux:**

```
pacman -Syu
pacman -S man-pages-zh_cn man-pages-zh_tw
```

**Red Hat / CentOS:**

```
yum update
yum install man-pages-zh-CN
```

**Fedora:**

```
dnf update
dnf install man-pages-zh-CN
```
