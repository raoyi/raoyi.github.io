admin 打开 PowerShell

```
winget search Microsoft.PowerShell
winget install --id Microsoft.PowerShell
或
winget install --id Microsoft.PowerShell --source winget
```

安装完成后，可使用 `pwsh` 启动 PowerShell7

使用 `powershell` 启动原有的 PowerShell5
